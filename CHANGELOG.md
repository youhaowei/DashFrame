## Versioning Strategy

DashFrame uses a hybrid versioning approach:

- **Web App** (`@dashframe/web`): Marketing-driven versioning where minor versions can include breaking changes
- **Library Packages**: Strict semantic versioning (breaking changes require major version bump)

See [docs/versioning.md](./docs/versioning.md) for details.
