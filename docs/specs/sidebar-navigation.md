# Sidebar Navigation Spec

The navigation is a sidebar rather than a header to give the main content more vertical space.

## Visibility

The sidebar is hidden. Can be toggled by the button attached to the center right of the sidebar. When hidden, the button is replaced by the chevron right icon attached to the center left of the screen.

## Collapsed/Expanded

The sidebar can be collapsed to show icons only or expanded to show icons and text. The toggle button is on the sidebar when the sidebar is visible.

## Dark/Light Mode

The sidebar has a theme toggle button, user can toggle between dark, light and system mode.
